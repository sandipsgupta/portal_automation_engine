=======================================================
  Portal Automation Engine — Quick Start Guide
=======================================================

DAILY WORKFLOW
──────────────
1. Download today's invoice sheet from Google Sheets
   File → Download → Microsoft Excel (.xlsx) or CSV

2. Save to any folder on your computer
   Example: D:\TATA_SHIPMENTS.xlsx

3. Double-click "Portal Automation Engine" on your Desktop

4. Click BROWSE → select today's sheet

5. Click ▶ RUN — the browser will open automatically
   and start entering invoice payments

6. Watch the Run Log for progress

7. When all entries are done, review the browser
   then press Enter to save to the portal

8. After the run → click "Export Log" → save the .txt
   file and send to your support team if anything looks
   unexpected


PORTAL SELECTION
────────────────
• Tata Consumer is currently active
• Other portals will be enabled in future updates


SETTINGS
────────
• Run headless  — hides the browser (not recommended)
• Debug mode    — adds extra detail to the log
                  (turn on only if asked by support)


SUPPORT
───────
If you see any errors, please:
1. Click "Export Log" and save the .txt file
2. Send the log file to your support contact
